## ② 概要説明（6ページ）

### BizOwnerとは？
- **副業感覚で会社経営 → 大家さん気分で権利収入を得るビジネス**
- 付属事業「ビジネスの学び家」（単体利用も可）
- **参加資格**：だれでも参加可能
- **知識**：簡単なので一切不要

### 詳細：3ステップで稼ぐ副業案件
1. **個人で出来る案件に参加**
2. **法人で出来る案件で追加収益**（※法人作成代行料の別途支払い必須）
3. **フランチャイズ化で更なる報酬を獲得**（※FC詳細は適正資格者のみ個別説明）

### 案件一覧

| 案件名 | 詳細 | 備考 |
|--------|------|------|
| 人材アカウント | 求人アカウントの運用代行 | インディード/エアワーク/エンゲージ/求人ボックス等 |
| ECショップ | 物販アカウントの運用代行 | メルカリ/楽天市場/shopee等 |
| 権利運用 | 法人所有の権利を共同運用 | ビザ事業/派遣事業/物販事業/レンタル事業など |
| 補助金/助成金 | 都度可能な案件を行政に申請 | 法人格 ※手出し無しなのでご安心を！ |
| 個人単発 | アフィリエイト/MNP/投資 | ここで得た報酬で法人を作成してください！ |
| 夢の事業 | あなたのやりたいビジネス！ | 学び家メンバー皆で全力支援！ |

### 報酬システム
- **権利収入**：一律1.5万円/月
- **単発**：案件によりけり
- ※FC加盟前の金額

### フランチャイズ制度
- 本件で作成した法人はあくまで「副業用」
  - FC前は運営の下請け/業務委託の関係（派遣社員と同じ扱い ≒ 雇われ社長）
- **フランチャイズ加盟で商流のトップに！**
  - 「ビジネスの学び家」オリジナルスキームを使いながら自分で自由に舵取り可能
  - FC化は「パクって独立」ということ

## ④ 参加フロー（4ページ）

### BizOwnerの進め方
1. 公式LINEの追加 / ウェビナーを全て確認
2. 個人情報＆決済フォームを入力
3. 会員サイトから個人案件を確認・実施
4. 法人作成を本部へ依頼し案件参加
5. 紹介活動でチーム作り / FC加盟で独立

### 紹介制度
- **会員紹介料**：
  - ① 人数×3,000円/月
  - ② 継続案件数×1万円/月
  - ③ 法人作成ボーナス：1万円
  - ④ フランチャイズ加盟ボーナス：加盟金の10%
- **紹介方法**：自身のアフィリエイトURLから会員登録のみ
- **報酬対象**：直紹介のみ（※二段目以降無し）

### Q&A
1. **作った法人は自由に使って良いですか？** → 原則禁止 / 本部が認めた事業のみ使用可能
2. **新しくサークルを作っても良いですか？** → 毎月1回開催「全体MTG」でプレゼン / 参加者多数で立ち上げ可能
3. **それぞれの案件やサービスの参加方法は？** → 全て「会員サイト」にて動画で説明
4. **個別の質疑応答はどこで出来ますか？** → 公式LINEに表示される「個別面談」へ
