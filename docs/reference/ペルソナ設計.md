# ペルソナ設計書

---

## 目的

本ドキュメントは、BizOwner / SARUDEMO / ビジネスの学び家の3サービスそれぞれについて、LP制作・広告設計・コピーライティングの判断基準となる**具体的なペルソナ**を定義するものである。

LP上のすべてのコピー・構成・CTA・デザイン判断は、このペルソナが「自分のことだ」と感じ、最終的にLINE登録するかどうかを基準に行う。

---

## ペルソナ設計の前提分析

### 自社サービスの分析

| 観点 | BizOwner | SARUDEMO | ビジネスの学び家 |
|------|----------|----------|-----------------|
| **提供価値** | 知識ゼロで始められる副業型オーナーシップ | 最短1年で社長になれる起業テンプレート | 学び×遊び×仲間が揃うコミュニティ |
| **価格帯** | 月額11,000円〜（低リスク・分割型） | 150万円一括（高投資・高リターン） | 月額11,000円（BizOwner会費に含む） |
| **決済条件** | 売上天引き可（CICブラック対応） | クレジットカード可（CICホワイト前提） | BizOwner入会が前提 |
| **心理変化** | 怪しい→自分でもできそう→LINE登録 | 不安→テンプレなら自分もいける→相談 | 面白そう→ここに居場所がある→参加 |
| **競合との差別化** | 信用ブラックでも参加可能な唯一の選択肢 | 実業FCテンプレ×プロチーム伴走 | 副業×起業×エンタメの複合型（他に類似なし） |

### 顧客層の分析

- **BizOwner層**：金融的に制約がある（CICブラック）ため、通常の起業塾やサブスク副業には参加できない。選択肢が限られている中で「月額11,000円・天引き可」は唯一の突破口。心理的には「起業」という言葉にビビるが、「副業感覚」「大家さん気分」という表現なら安心できる。
- **SARUDEMO層**：信用も資金もあるが、起業の具体的な進め方が分からず足踏みしている。「いつか」ではなく「今」動きたい。150万円を投資する覚悟はあるが、再現性の証拠がないと踏み切れない。
- **学び家層**：ビジネスだけでは物足りない。「稼ぐ」と「楽しむ」を両立したい。都会で孤独を感じている若手が、同世代の仲間と繋がれる場所を探している。

## ペルソナの活用方法

### LP制作時の判断基準

すべてのLP制作判断において、以下の問いを立てること：

1. **このコピーを読んで、ペルソナは「自分のことだ」と感じるか？**
2. **このデザイン・トーンは、ペルソナの日常で触れているメディアと違和感がないか？**
3. **このCTAのハードルは、ペルソナが「今すぐ」行動できるレベルか？**
4. **ペルソナの最大の不安・疑問に対して、LP上で回答できているか？**

## PDCAの運用方針

### 検証項目

- LP公開後のCVR（LINE登録率）をペルソナごとに計測
- 広告のクリック率・直帰率からペルソナとの整合性を検証
- LINE登録後の実際の顧客属性とペルソナの乖離を分析

### 更新タイミング

- LP公開後1ヶ月時点でデータを確認し、ペルソナの修正要否を判断
- 四半期ごとにペルソナの見直しを実施
- 大きなサービス変更（料金改定・ターゲット変更等）があった場合は即時更新

### 注意事項

- ペルソナは「こうあってほしい顧客」ではなく「実際に来る可能性が最も高い顧客」を描くこと
- 思い込みで作らず、実際の問い合わせ・成約データが蓄積されたらペルソナを修正すること
- 1つのサービスに対して1人のペルソナに絞り込むこと。「あの層もこの層も」と広げるとLPの訴求がぼやける
