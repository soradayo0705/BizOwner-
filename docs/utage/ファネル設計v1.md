## 1. 前提・確定事項

### 案件スコープ
- **依頼内容**：UTAGE構築 + 3サービス分のLP制作
- **依頼外**：UTAGE運用・LINE個別対応・広告運用
- **方針**：UTAGEで賄えるものは原則UTAGE内で構築。賄えないものはクライアントへ確認

### サービス構成
| サービス | 立ち位置 | 料金 |
|---|---|---|
| **BizOwner** | CICブラック向け副業 | 月額11,000円 + 法人作成10万円 + FC加盟200万円（段階課金） |
| **SARUDEMO** | CICホワイト向け本格起業 | 契約金150万円（一括） |
| **ビジネスの学び家** | BizOwner/SARUDEMOの福利厚生コミュニティ＋単独入会も可 | 月額10,000円（追加課金） |

### 学び家の取り扱い
- **BizOwner / SARUDEMO 加入者でも、学び家に参加する場合は別途月額1万円が発生**
- 学び家単独の入会も可能 → **学び家LPも独立して制作する**
- 決済フォームで「学び家も追加する」を選択できる仕組みにする

### 個別面談の位置付け（資料根拠あり）
> `docs/reference/資料まとめ.md:138`
> **「個別の質疑応答はどこで出来ますか？」→「公式LINEに表示される『個別面談』へ」**

> `docs/reference/資料まとめ.md:42`
> **「FC詳細は適正資格者のみ個別説明」**

- **入会前の面談は必須ではない**
- LINE登録後、**質問のある人だけが任意で予約**する導線
- **FC加盟だけは個別説明必須**（適正者向け）

## 3. 各UTAGEブロックで「何を作るか」具体イメージ（学び家追加対応版）

### ① LINE・ステップ配信
| 作るもの | 中身 |
|---|---|
| あいさつメッセージ | 登録直後の自動返信＋アンケートURL |
| 初回アンケートフォーム | UTAGEフォームで作成→回答に応じてタグ自動付与（信用情報・興味サービス・本気度・支払い意向・学び家興味度） |
| リッチメニュー（4〜6マス） | サービス紹介／ウェビナー／**任意の個別面談予約**／よくある質問／運営者プロフィール／学び家紹介 |
| ステップ配信シナリオ | Day1〜Day7の自動配信文。**source別 × アンケート別**にシナリオ分岐 |
| 学び家オプション訴求配信 | BO/SARUシナリオの中盤に「学び家も追加するとこれが手に入る」訴求を組み込む |
| ABテスト系統 | 配信文・サムネ・件名を2系統並走 |
| 単発配信 | キャンペーン・締切告知用 |

### ② ウェビナー
**= 録画動画を「ライブ風に予約配信」する仕組み**

| 作るもの | 中身 |
|---|---|
| BizOwnerウェビナー動画 | 顧問登壇動画（資料①〜⑤ベース） |
| SARUDEMOウェビナー動画 | 顧問登壇動画（SARUDEMO資料ベース） |
| 学び家紹介動画 | 福利厚生・サークル・イベントを紹介する短尺動画（5〜15分） |
| 申込フォーム | 各ウェビナー個別に作成。日時選択＋メアド取得 |
| リマインドメール/LINE | 当日／1時間前／開始直後 |
| 視聴ページ | 視聴ログ計測（離脱ポイント分析可） |
| 視聴後CTA | 動画下に「決済フォームへ」または「個別面談予約」ボタン |

### ③ 決済
| 作るもの | 中身 |
|---|---|
| BizOwner決済フォーム | 月額11,000円サブスク＋初月無料／**「学び家も追加する」チェックボックス（+月10,000円）** |
| BizOwner法人作成決済 | 10万円スポット決済（メンバーサイト内から呼出） |
| BizOwner FC加盟決済 | 200万円スポット決済（個別面談後にURL送付） |
| SARUDEMO決済フォーム | 150万円一括（クレカ可）／**「学び家も追加する」チェックボックス（+月10,000円）** |
| 学び家単独決済フォーム | 月額10,000円サブスク（学び家LP経由の単独入会者用） |
| 2期目以降切替処理 | 1年経過後の月額10万円顧問料への自動切替（要設計確認） |

### ④ メンバーサイト（会員サイト）
**= ログイン後の動画・申込ボタンが並んだ会員専用サイト**

```
┌─ メンバーサイト構造 ────────────────────────┐
│                                              │
│ 【共通エリア】                               │
│  ・マイページ（会員ランク・紹介URL表示）     │
│  ・お知らせ                                  │
│                                              │
│ 【BizOwner会員のみ表示】                     │
│  STEP1：個人案件動画                         │
│   ├─ オリエンテーション                      │
│   ├─ 人材アカウント運用                      │
│   ├─ ECショップ運用                          │
│   └─ 補助金/助成金                           │
│  STEP2：法人案件【¥100,000で解放】          │
│   └─ [法人作成を申し込む]ボタン              │
│  STEP3：FC加盟【適正者のみ】                 │
│   └─ [FC個別面談予約]ボタン                  │
│                                              │
│ 【SARUDEMO会員のみ表示】                     │
│  起業フルパッケージ動画群                    │
│  資金調達サポート                            │
│  顧問チームへの相談予約                      │
│                                              │
│ 【学び家会員のみ表示】※BO/SARUと別フラグ    │
│  ├─ 月次オンラインセミナー動画               │
│  ├─ 占い・コーチング予約                     │
│  ├─ サークル参加申込                         │
│  ├─ イベント・コンテスト情報                 │
│  └─ プレゼント・特別優待券                   │
│                                              │
│ 【紹介プログラム（全員）】                   │
│  ├─ あなたの紹介URL                          │
│  ├─ 紹介人数・報酬実績                       │
│  └─ 紹介マニュアル動画                       │
└──────────────────────────────────────────────┘
```

→ **UTAGE側の作業：「コンテンツ陳列棚＋決済ボタン＋紹介URL自動発行」を作るだけ**
→ 案件の中身・法人作成・FC審査はクライアント側の人間オペレーション

### ⑤ アフィリエイト（紹介機能）
| 作るもの | 中身 |
|---|---|
| 紹介URL自動発行 | 会員1人ずつにユニークURL付与 |
| 成果計測 | 紹介経由の登録/決済を自動カウント |
| 報酬計算ルール設定 | 人数×3,000円・継続案件×1万円・法人作成1万円・FC加盟金10% |
| 紹介者用ダッシュボード | マイページに紹介実績表示 |

### ⑥ 個別面談予約（任意）
- UTAGEのカレンダー機能で予約枠管理
- ZOOM連携で自動的に会議URL発行
- 事前ヒアリングフォーム（信用情報・現状・支払い意向）
- リマインド自動配信
- **位置付け：必須ではなく、質問者・FC希望者のみが予約**

## 4. Phase 1：BizOwnerファネル簡易版（最小構成）

> 目的：BizOwner一本で「LINE登録→決済→メンバーサイト入室」までの骨組みを最速で作り、クライアントに見せて方向性を確認する

### Phase 1で作るもの

#### ① LINE・ステップ配信【簡易版】
| 作るもの | 簡易版の中身 |
|---|---|
| あいさつメッセージ | 1パターンのみ。アンケートURL付き |
| 初回アンケート | 4〜5問のみ（信用情報・興味サービス・本気度・学び家興味・備考） |
| リッチメニュー | 4マス（ウェビナー／個別面談予約／よくある質問／運営者プロフィール） |
| ステップ配信 | **BizOwner用1系統のみ**。Day1〜Day5の5通（共感→課題→解決策→事例→CTA） |
| ABテスト | **Phase 1ではなし**（Phase 2以降で実装） |

## 5. 進め方フェーズ

```
Phase 1: BizOwner簡易版を構築 ★今ここ
   ↓ クライアント確認・修正
Phase 2: SARUDEMOファネル追加（Phase 1テンプレ流用）
   ↓ クライアント確認・修正
Phase 3: 学び家LP経由の単独入会導線追加
   ↓
Phase 4: ABテスト並走・シナリオ分岐の高度化
   ↓
Phase 5: メンバーサイト実コンテンツ・紹介報酬自動化・FC加盟導線
   ↓
Phase 6: 統合テスト・本番リリース
```

**理由（並列ではなく逐次にする根拠）：**
1. 同じ箱を3つ作る無駄を防げる（1つ固まればテンプレ流用で爆速）
2. クライアントの認識ズレを早期発見できる
3. 修正コストが大幅に下がる
4. UTAGEの構造理解が進み、見積もり精度が上がる

## 6. UTAGE単独で完結できないもの（クライアント要対応）

| 項目 | UTAGEで出来ない部分 | 代替/補完 |
|---|---|---|
| 法人作成 | 設立手続きそのもの | 本部の司法書士/行政書士オペ |
| 案件実施 | 各案件の運用作業 | 外部ツール（メルカリ等） |
| LINE個別返信 | 手動カスタマー対応 | 運営チームの人手 |
| BizOwner売上天引き | 売上から月額11,000円を引く処理 | UTAGE決済（クレカ）or 別途経理処理 |
| 学び家のリアルイベント運営 | 当日運営 | 別途運営チーム |
| ZOOMアカウント | 会議そのもの | 別途ZOOM契約 |
| ウェビナー動画の撮影 | 顧問の登壇撮影 | クライアント側で撮影 or 別発注 |
