## 2. 参考記事の知識ベース

参考URL：https://stock-sun.com/column/lp-configuration/
タイトル：「LP(ランディングページ)の基本構成とは？制作会社しか知らない重要な要素と注意点」
著者：深井嵐丸（StockSun株式会社 LP職人）

### 2-1. そもそもLP（ランディングページ）とは？

LPは、特定の目的に応じて訪問者に行動を促すために作成されるウェブページのこと。主な目的は **商品やサービスの購入、問い合わせの獲得、会員登録** など。

特徴：
- 基本的には長めのページとして構成され、**ページを跨がずに1ページのみで完結**する
- 読んでもらいたい順番通りにストーリーを伝えられる
- toB、toC両方で活用される
- 広告やSNSなどのリンクから直接訪問者を誘導し、最終的なコンバージョンを目指す
- 構成と内容の分かりやすさ・見やすさが命

### 2-2. LPの3つの基本構成

LPは構成が命。基本構成は大きく3つ。これさえ覚えれば、あとは要素を足していくだけ。

| 構成 | 役割 |
|------|------|
| ① ファーストビュー | ユーザーが一番知りたい情報を記載する箇所 |
| ② ボディ | ユーザーにとって役立つ理由を記載する箇所 |
| ③ クロージング | 購入や申し込みを促す箇所 |

#### ① ファーストビュー

- LPを訪れたユーザーが最初に目にする領域
- ここで興味を持ってもらえないと離脱に繋がる
- ユーザーが最も興味を持つ情報や、提供する価値を簡潔に伝えるキャッチコピーを配置
- ターゲットユーザーに向けた最適なCTAボタンを設置し、スムーズに次のステップへ誘導
- **LPにとって一番重要であり、もっとも時間をかけるべきポイント**

#### ② ボディ

- LP構成の中心部分
- ユーザーがなぜその商品やサービスを利用すべきかを理解するための情報を提供
- 製品やサービスの特徴、ベネフィット、ユーザーの課題解決にどう役立つかを説明
- **「興味がある」→「実際に使いたい！」という心理状態を作り上げる**
- 顕在化されているニーズだけでなく、**潜在的なニーズにもアプローチ**する必要がある
- 事前リサーチでユーザー理解を深めることが大切

#### ③ クロージング

- LP構成の最終段階
- ユーザーに行動を促すエリア
- 明確なCTAを用いて、購入や問い合わせ、資料請求などの行動を促す
- 業界やターゲットの性質に合わせて訴求方法を変える
- **明確なメリットがあるとクリックしてもらいやすくなる**

### 2-3. LPの構成に必要な7つの基本要素

| # | 要素 | 配置場所 |
|---|------|---------|
| 1 | キャッチコピー・アイキャッチ画像 | ファーストビュー |
| 2 | CTAボタン | ファーストビュー |
| 3 | 導入文 | ボディ |
| 4 | 商品・サービス情報 | ボディ |
| 5 | 実績・お客様の声 | ボディ |
| 6 | Q&A・サービス導入の流れ | クロージング |
| 7 | 入力フォーム | クロージング |

#### 要素① キャッチコピー・アイキャッチ画像

- ユーザーの中にはここで離脱してしまう人も多い
- **簡潔にかつ分かりやすく作成する**
- 数字を入れることで具体性が上がる

**キャッチコピー**：
- ユーザーに強く響くフレーズ
- 同時に共感や想起をさせることができるとベスト
- 商品やサービスのベネフィットを端的に伝え、その価値を感じさせる文章にする
- 売れているLPでは100案以上から精査することも珍しくない

**アイキャッチ画像**：
- サービスのベネフィットを視覚的に伝える要素
- ユーザーの関心を引き、サービスの利便性や効果を視覚的に伝える
- 安心感を与えるものや、緊急性を煽るものなど、商材によって使い分ける

#### 要素② CTAボタン

- ファーストビューにCTAを配置し、ユーザーに次のアクションを促す
- ちょっと動くようなものや目立つデザイン
- 心理学に基づいたアプローチも有効
- **今すぐアクションを起こしたいユーザーの取りこぼし防止**につながる

#### 要素③ 導入文

- LP全体の概要を説明し、ユーザーが続きを読みたくなるように構成
- ユーザーがどんな悩みを持っているかにフォーカスして共感する
- **「このLPが自分に合っている！」という認識を持たせる**
- 悩みの解決がLPでは大切

#### 要素④ 商品・サービス情報

- 商品やサービスの詳細を具体的に記載
- なぜ悩みを解決できるかを具体的に説明
- **ふわっとしているイメージをリアルなものに固める**
- できるだけ細かく、説明のし忘れがないように網羅的に記載

#### 要素⑤ 実績・お客様の声

- ユーザーに対して信頼感を与える重要な要素
- 実際の改善例や顧客の成功事例を通じて、サービスの効果を証明
- **ユーザーに安心感を提供する**
- 具体的に書かれたものや、悩みの母数として多いものを選ぶ
- 顔出しできるとさらに信頼度が上がる

#### 要素⑥ Q&A・サービス導入の流れ

- ユーザーの疑問を解消する
- よくある質問と回答、サービス導入までの具体的な流れを示す
- **ユーザーが安心して問い合わせや申し込みに進めるようにする工夫**
- 解約に関してなど不安要素を解消する部分

#### 要素⑦ 入力フォーム

- 最終的な行動を促すために設置
- シンプルで入力しやすいフォームを配置
- **入力する項目数はできるだけ少ない方が、心理的なハードルは下がる**
- LPの構成やデザインが完璧でも、入力フォームが悪いとCVRを大きく悪化させる

### 2-4. ワイヤーフレームの作り方

LPを構築する前にワイヤーフレームを作成する工程がある。事前作成によってLPの全体像がつかめ、作成時のイメージが掴みやすくなる。

#### 工程① レイアウトを決める

- ユーザーがスムーズにLP内を移動できるように設定
- **ユーザー目線に立って、どういった心理状態で見るのかを客観視**
- どういった流れで興味を持ってもらい、最終的に申し込んでもらうかを大まかに決める
- 競合や理想とするLPを参考にしながら進める

#### 工程② レイアウトに要素を当てはめる

- レイアウトの枠組みに要素を適切に配置し、LP全体の統一感と効果を高める
- 心理学や業界の傾向を考慮するとより効果的

### 2-5. LP構成・作成の7ステップ

| ステップ | 内容 |
|---------|------|
| ① | KGI・KPIを決める |
| ② | ペルソナを分析する |
| ③ | 構成（ワイヤーフレーム）を作成する |
| ④ | ライティングを行う |
| ⑤ | デザインを作成する |
| ⑥ | コーディング作業を行う |
| ⑦ | 公開する（LPO） |

#### ステップ① KGI・KPIを決める

- 効果を最大化するLP構成を設計するには、目標となるKGIやKPIを明確に設定することがスタート
- どのくらいのCVが必要か、そのためにどのくらいのインプレッション数があればいいか
- KPIに基づいた効果的なLP改善を繰り返してベストなLPに仕上げていく

#### ステップ② ペルソナを分析する

- ターゲットユーザーを明確にする
- どんな属性で、どんなデザインを好むのか、顕在ニーズや潜在ニーズを洗い出す
- **ターゲットユーザーに最適化されたコンテンツを提供するためのペルソナ分析を徹底**
- ペルソナによってLP構成も大きく変わるため、できる限り詳細にかつ正確に洗い出す

#### ステップ③ 構成（ワイヤーフレーム）を作成する

- LPの全体構成を考え、ユーザーが自然に行動を起こせるレイアウトを決定
- クリエイティブの制作はこの段階では行わないが、**どんな内容のクリエイティブを挿入するかは決めておく**
- 箇条書きで要点をまとめておくと、完成後の流れやユーザー目線での見直しができる

#### ステップ④ ライティングを行う

- LPのコンテンツを具体化する重要なフェーズ
- 構成に沿って的確な訴求を行うライティング能力が求められる
- **クリエイティブ（画像）もこの段階で作成**
- 全体のバランスを考えながら作成すると完成時の違和感がなくなる

#### ステップ⑤ デザインを作成する

- ユーザーの目に入りやすいデザインで読了率を上げる
- **おしゃれなデザインだけでなく「CVRの高いデザインかどうか」のジャッジも必要**
- ペルソナと見比べながら最適なデザインを構築

#### ステップ⑥ コーディング作業を行う

- デザインに合わせたコーディングを行い、実際のLPを構築
- **編集しやすいコーディングを組めるかどうかをあらかじめ相談**
- 修正することを想定しながら構築すると、後々のLP運用がしやすい

#### ステップ⑦ 公開する（LPO）

- LP公開後も継続的に効果を高めるための最適化（LPO）が必要
- ヒートマップ分析などでデータを基に定期的に修正
- **中長期的にどう改善していくかも含めて伴走**することが理想

### 2-6. LP構成を作る上での3つのポイント

#### ① ユーザー・お客様目線で伝える

- ユーザー視点を忘れず、メリットをわかりやすく伝える
- ユーザーに共感される内容を提供することで高いコンバージョンを実現

#### ② 要点をわかりやすくする

- 複雑な情報は避け、要点を簡潔に伝える
- LPは読者が理解できない・難しいと感じてしまうとすぐに離脱される
- 構成とライティングで要点をわかりやすく説明する力が求められる

#### ③ 複数の構成を作りABテストを行う

- ABテストにより最適な構成を見つけ出し、CVRを最大化
- データに基づくテストを実施し、継続的な改善を行う

### 2-7. LP構成を考える上での7つの注意点

#### ① LPと広告を連動させる

- LPと広告内容の一致は、ユーザーの信頼感を高める重要なポイント
- 広告経由でも内容と一致しないと離脱につながる
- 広告とLPの連動を意識し、最適なメッセージを提供

#### ② BtoBは「投資」、BtoCは「消費」を意識する

- **BtoBでは「投資」的な意識**を持って作成し、使用する利益があると判断してもらう
- 投資に対してどのくらいの効果が見込めるかを重点的にアピール
- **BtoCの場合は「消費」を意識**し、LP内で購買までを完結させる

#### ③ 事前リサーチを行う

- ユーザーに最適化されたコンテンツを提供する上で欠かせない
- 事前リサーチが不十分だと効果の出ないLPになる

#### ④ ABテストを繰り返す

- ABテストの繰り返しでLPの精度を上げることはLP制作の常識
- データを基にした改善を行い、CVR向上を目指す
- どこを変えたらどういった数値になったかを記録して着実に成果を出す

#### ⑤ 他社事例を真似しすぎない

- 他社との差別化を意識したLPや構成を作成
- **独自性を保つことが、信頼性と効果を高めるポイント**

#### ⑥ 法律や媒体規定に抵触しない

- 法的規範に基づき、安心して利用できるLPを制作
- 薬機法などで心配な場合は専門家に確認
- 事前に法律に抵触しないかをチェック

#### ⑦ 改善を繰り返せる設計にする

- LPは改善をしていくことで最大の効果を発揮
- 最初から上手くいくことはほとんどない
- 改善案が出たらすぐに修正できるような設計にしておく

## 3. 今回のBizOwner LPへの適用方針

参考記事の理論を、今回のBizOwnerに具体的にどう落とし込むかを記載。

### 3-1. ステップ① KGI・KPI

| 指標 | 内容 |
|------|------|
| KGI | 公式LINE友だち追加数 |
| サブKPI① | FV到達率 |
| サブKPI② | スクロール率（セクション6到達率） |
| サブKPI③ | CTAクリック率 |

**LINE登録をゴールにする理由**：
- BizOwner運営側に「LINE→ウェビナー→決済フォーム」の導線が既に整備済み
- 月額1.1万円のサブスク契約をLP初訪問で決断させるのはハードルが高い
- 「1ヶ月無料」と組み合わせて軽い一歩を促すのが合理的

### 3-2. ステップ② ペルソナ

資料から読み取れる想定読者：

- **年齢**：20〜30代中心
- **属性**：副業や起業に興味はあるが知識ゼロ〜浅い
- **悩み**：「会社員のままじゃ嫌」「でも何から始めれば分からない」
- **性格**：仲間とワイワイやりたい気質、エンタメ要素に反応する
- **その他**：CICブラックでも参加OKなので、ローン審査が通らない層も含む

→ このペルソナだと、**「怖そう・難しそう」という警戒心を解く**ことが最優先。「大学のサークル的なノリ」というクライアント要望は、このペルソナに対して非常に正しい方向性。

### 3-3. ステップ③ 構成（このドキュメント）

→ 次セクションの「ワイヤーフレーム全体構成」へ

## 5. 各セクション詳細

### Section 01 — ファーストビュー

| 項目 | 内容 |
|------|------|
| 役割 | 参考記事の「①キャッチコピー・アイキャッチ画像」+「②CTAボタン」 |
| 目的 | 3秒で「自分向けだ」「面白そう」と思わせ、CTAに誘導 |
| 主要コピー | 「副業感覚で、会社のオーナーになる。」 |
| サブコピー | 「月額1.1万円から始める、あなたの権利収入ライフ。」 |
| アイキャッチ | 資料PDFの女性ビジュアル（六角形クリップ） |
| CTA | 「1ヶ月無料でLINE登録する」 |

**何をやるか**：LPの命であるFV。読者がここで離脱しないよう、メインキャッチで掴み、サブキャッチで補足し、すぐ下のCTAでアクションへ誘導。背景はダークネイビーで六角形パターンを薄く散らす。

### Section 02 — 共感パート（Problem）

| 項目 | 内容 |
|------|------|
| 役割 | 参考記事の「③導入文」 |
| 目的 | 読者の悩みに共感し、「これは自分の話だ」と思わせる |
| 構成 | 5つの悩みカード |

**悩みリスト**：
1. 副業を始めたいけど、何から手をつければいいか分からない
2. ビジネスの知識がゼロで、失敗するのが怖い
3. 一人だとモチベーションが続かない
