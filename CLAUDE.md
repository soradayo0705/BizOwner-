# BizOwner プロジェクト CLAUDE.md

---

## プロジェクト概要

本プロジェクトは、BizOwnerを運営する会社の**コーポレートサイト（HP）**と、その配下に紐づく**3つのサービスLP**を制作するものである。

---

## サイト構成

```
コーポレートサイト（HP）
├── トップページ
├── 会社概要
├── サービス一覧
│   ├── BizOwner（LP） ← 詳細ページ
│   ├── SARUDEMO（LP） ← 詳細ページ
│   └── ビジネスの学び家（LP） ← 詳細ページ
├── お問い合わせ
└── ...
```

### 導線設計
1. コーポレートサイト（HP）をまず1つ作成する
2. HPの「サービス一覧」から各サービスを選択すると、それぞれのLPへ遷移する
3. 各LPはサービスごとに独立した詳細ページとして制作する

---

## サービス構成と関係性

### BizOwner と SARUDEMO の関係
- **実質的には同じサービス**。ターゲット顧客に合わせて2つに分けて提供している
- **BizOwner**：CICブラックの方が対象（年齢・知識不問）、売上から天引きで支払い可能
- **SARUDEMO**：CICホワイトの方が対象、クレジットカード払い可能
- どちらも「副業感覚で法人を持ち、権利収入を得る」ビジネスモデル

### ビジネスの学び家
- BizOwner / SARUDEMO とは**別サービス**（総合コミュニティ）
- 詳細はHP + BizOwner LPの制作・デザインが完成し次第、改めて詰める

## 制作物一覧

### 1. コーポレートサイト（HP）
- 会社の顔となるサイト
- サービス一覧ページを設け、各LPへの導線を作る

### 2. BizOwner LP
- ターゲット：CICブラック / 副業初心者 / 知識不要で始めたい層
- 訴求軸：副業感覚で会社経営、大家さん気分で権利収入
- 3ステップ（個人案件→法人案件→FC化）の分かりやすさを重視
- 費用：月額11,000円 + 法人作成10万円 + FC加盟200万円

### 3. SARUDEMO LP
- ターゲット：CICホワイト / 本格的に起業したい層
- 訴求軸：「猿でも出来る起業」最短最速の起業ファストパス、年商1億円を目標
- 費用：契約金150万円（一括）+ 学び家会費 月額1万円
- 報酬モデルがBizOwnerより大きい（権利収入30〜100万/月等）

### 4. ビジネスの学び家 LP
- BizOwner / SARUDEMO とは別サービス
- **HP + BizOwner LPのデザイン完成後に詳細を詰める（現時点では保留）**

## LP制作ステップ

各LPは以下の5ステップで制作する：

1. **STEP1. 誰に届けるか（ターゲット設計）**
2. **STEP2. 何をさせたいか（ゴール設計）**
3. **STEP3. どんな順番で伝えるか（構成）**
4. **STEP4. どうやって信頼を積むか（証拠）**
5. **STEP5. どこで行動させるか（CTA）**

## STEP1. ターゲット設計（誰に届けるか）

### BizOwner LP

| 項目 | 内容 |
|------|------|
| **信用情報** | CICブラック（クレジットカード審査が通らない方） |
| **年齢・知識** | 不問。完全未経験OK |
| **現在の状況** | 本業の収入だけでは不安／副業を始めたいが何をすればいいか分からない |
| **求めていること** | 知識ゼロ・リスク最小限で始められる副収入の仕組み |
| **支払い能力** | まとまった初期投資は難しい。売上からの天引き払いなら可能 |
| **心理的ハードル** | 「起業」という言葉に尻込みする／自分にできるか不安／騙されたくない |

**ペルソナ像**：20〜40代、正社員 or フリーター。副業に興味はあるが知識もコネもない。CICブラックのため通常の金融サービスやクレカ決済ができず選択肢が限られている。「簡単に」「リスクなく」始められるなら挑戦したいと思っている。

### SARUDEMO LP

| 項目 | 内容 |
|------|------|
| **信用情報** | CICホワイト（クレジットカード決済が可能） |
| **年齢・知識** | 不問だが、本気で起業・脱サラを志す層 |
| **現在の状況** | 会社員として働いているが独立願望が強い／既に副業経験があり次のステップを探している |
| **求めていること** | 最短最速で起業し、年商1億円規模のビジネスオーナーになるロードマップ |
| **支払い能力** | 契約金150万円を一括で支払える（クレジットカード可）。資金調達サポートも活用可能 |
| **心理的ハードル** | 150万円という投資額に見合うリターンがあるか不安／本当に再現性があるのか疑問 |

**ペルソナ像**：20〜30代、会社員 or 既に副業で実績あり。「いつかは独立したい」ではなく「今すぐ動きたい」温度感。起業に対して前向きだが、一人では何から始めればいいか分からない。投資意欲はあるが、テンプレート・サポート体制がないと踏み切れない。

全LPのメインCVは**公式LINE追加**に統一する。LINE追加後の導線（ウェビナー視聴・個別面談・契約）はLINE側で設計する。

|------|------|
| **メインCV** | 公式LINE追加 |
| **LP上での心理変化** | 「怪しい…」→「自分でもできそう」→「とりあえずLINE追加してみよう」 |

## LP制作ガイドライン

> 参考：https://stock-sun.com/column/lp-configuration/

### LPとは
- 特定の目的（購入・問い合わせ・会員登録）に向けてユーザーに行動を促す1ページ完結型のWebページ
- 広告・SNSからユーザーを直接誘導し、CVRを最大化することが目的
- 構成と内容の分かりやすさ・見やすさが成否を左右する
- **LPの出来はCVRに直結する**

### 基本構成の3ブロック

#### 1. ファーストビュー（最重要）
- ユーザーが最初に目にする領域。**ここで興味を持たせないと即離脱**
- 最も時間をかけるべきパート
- 伝えること：ユーザーが最も知りたい情報／提供価値のキャッチコピー
- CTAボタンを設置し、すぐ行動したいユーザーの取りこぼしを防ぐ

#### 2. ボディ（中心部）
- 「なぜこのサービスを使うべきか」を理解させるエリア
- 顕在ニーズだけでなく**潜在ニーズにもアプローチ**する
- 目指す心理変化：「興味がある」→「実際に使いたい！」
- 事前リサーチによるユーザー理解が土台になる

#### 3. クロージング（最終段階）
- 行動（購入・問い合わせ・資料請求）を促すエリア
- 明確なCTAとメリット訴求でクリック率を高める
- 業界・ターゲット特性に応じて訴求方法を変える

### 7つの基本要素

| # | 要素 | 配置場所 | ポイント |
|---|------|----------|----------|
| 1 | キャッチコピー＋アイキャッチ画像 | ファーストビュー | ベネフィットとオファーを端的に。数字で具体性UP。100案以上から選ぶ覚悟で |
| 2 | CTAボタン | ファーストビュー | 動くデザインや目立つ色で今すぐ行動を促す |
| 3 | 導入文 | ボディ | ユーザーの悩みに共感→「このLPは自分向けだ」と思わせる |
| 4 | 商品・サービス情報 | ボディ | 具体的・網羅的に記載。サービス利用後のイメージを固める |
| 5 | 実績・お客様の声 | ボディ | 具体的な成功事例で信頼感を醸成。顔出しがあるとさらに効果的 |
| 6 | Q&A＋導入の流れ | クロージング | 不安要素（解約など）を解消し、申し込みへの心理ハードルを下げる |
| 7 | 入力フォーム | クロージング | **入力項目は最小限に**。フォームのUXがCVRを左右する |

### ワイヤーフレームの作り方
1. **レイアウトを決める** — ユーザーの心理状態・閲覧の流れをシミュレートし、競合LPや参考LPを見ながら構成の骨格を決める
2. **レイアウトに要素を当てはめる** — 心理学・業界傾向を踏まえて要素を配置し、統一感と効果を両立させる

### LP制作の7ステップ

1. **KGI・KPIを決める** — 必要CV数・インプレッション数などを逆算して設定
2. **ペルソナを分析する** — 属性・好むデザイン・顕在/潜在ニーズを詳細に洗い出す
3. **ワイヤーフレームを作成する** — 箇条書きで要点まとめ→ユーザー目線で見直し
4. **ライティングを行う** — 構成に沿って的確な訴求文を書く。画像制作も並行
5. **デザインを作成する** — 「おしゃれかどうか」より「CVRが高いか」で判断
6. **コーディングを行う** — 修正を想定した編集しやすいコーディング。後のLPO運用を見越した設計
7. **公開 → LPO（継続的改善）** — ヒートマップ等のデータを基に定期的に修正

### 構成を作る上での3つのポイント
1. **ユーザー・お客様目線で伝える** — メリットをわかりやすく、共感を軸に構成
2. **要点をわかりやすくする** — 複雑な情報は避ける。難しいと感じた瞬間に離脱される
3. **複数の構成を作りABテストを行う** — データに基づいて改善を繰り返しCVRを最大化

### 構成作成時の7つの注意点

1. **LPと広告を連動させる** — 広告とLPの内容不一致は即離脱→広告費がムダに
2. **BtoB vs BtoCの意識の違い** — BtoB=「投資」ROI訴求 / BtoC=「消費」LP内で購買完結
3. **事前リサーチを徹底する** — リサーチ不足のLPは効果が出ない
4. **ABテストを繰り返す** — 改善記録を残す。LPは公開して終わりではなく改善し続けるもの
5. **他社事例を真似しすぎない** — 独自性が信頼性と効果を高める
6. **法律・媒体規定に抵触しない** — 薬機法など業界に応じた法律チェック必須
7. **改善を繰り返せる設計にする** — 修正しやすいコーディング設計が機会損失を減らす
